# 스킬카드 설계

## 컨셉
기괴하고 저주받은 기생 무기 + 소모품 숙주.
카드는 무기의 세 부위(칼날/코등이/손잡이)에 대응하며,
각각 "공격 / 생존 / 자원 착취"라는 역할을 담당한다.

**리소스 축:**
```
HP 소모 ←→ 스트레스 소모
공격력  ←→ 생존력
```

## 구현 메모

- 모든 카드는 현재 `_executeCard`의 `aoe / dash / heal / calm` 타입으로 구현 가능
- 이펙트 파일은 `BattleScene.preload()`에 spritesheet로 추가 필요
- Pack #7은 단일 PNG가 아닌 GIF 파일 → 별도 처리 방식 검토 필요
- Pack #8 Shield는 다중 상태 PNG (Start/Idle/Break) → 상태 전환 로직 필요
- `hpCost` 음수 = HP 회복, `stressCost` 음수 = 스트레스 감소 (기존 `applyCardCosts` 그대로 적용)
